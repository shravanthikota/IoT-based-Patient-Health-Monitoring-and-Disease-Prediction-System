Data Cleaning Process

How did we clean our data?

The dataset had few rows with empty strings. We convert empty string to NaN

 df=pd.read_csv('dataset_new.csv')
 df_fullna = df_full.replace('?', np.nan)

Now we fill in the NaN values to median for continuous variable and 1 for catagorical variables values and do some cleaning.


For cleaning the data: 
df_fullna.isnull().sum()
df = df_fullna
df = df.convert_objects(convert_numeric=True) #turn data into numeric type for computation
df['vessels'] = df['vessels'].fillna(df['vessels'].median())
df['thal'] = df['thal'].fillna(df['thal'].median())
df.isnull().sum()
df.loc[:, "diagnosis"] = df.loc[:, "diagnosis"].apply(lambda x: 1 if x >= 1 else 0)
df2=df

Spliting the data for Training and Testing 
X_train, X_test, y_train, y_test = train_test_split(df2.iloc[:,:-1], df2['diagnosis'], test_size = 0.33, random_state=44, stratify= df2['diagnosis'] )

Algorithm

Which algorithms are used and why?

Random Forest
Random Forests perform implicit feature selection and provide a pretty good indicator of feature importance. Random Forests are very quick to train. It's a stroke of brilliance when a performance optimization happens to enhance model precision, or vice versa.
